# Prism Memory

Prism MCP support is optional.

When enabled, use it as a semantic memory and handoff ledger for the current workspace/project. Keep it separate from secrets and validate every remembered claim against live state.

Suggested usage when tools are available:

- load context at session start
- search memory during planning
- save handoffs or durable decisions at session end

Do not require Prism for the workspace to function.
